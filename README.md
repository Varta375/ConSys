# ConSys 

Утилита мониторинга Acer Nitro 5 AN515-45 (Ryzen 5 5600H RTX 3060)

## Описание

Утилита для мониторинга температур, управления вентиляторами,
режимами питания и процессами.  
Разработана для ноутбука Acer Nitro 5 AN515-45

Программное обеспечение предоставляется "как есть" без какой-либо гарантии, явной или подразумеваемой.
Неправильное использование (в том числе попытки модификации кода программы) может привести к нестабильности или неисправности системы

## Функционал

Мониторинг температур CPU/GPU

Управление вентиляторами (Макс/Вручную)

Переключение схем электропитания

Остановка процессов

Ограничение максимальной температуры процессора

## Сборка из исходников

### Требования
CMake 3.20+
Qt 6.5+ (MSVC 64-bit)
Visual Studio 2022 (с C++ и CMake)

### Установка
Первый вариант: перейти в раздел Releases и скачать оттуда zip-папку

Второй вариант, склонируйте репозиторий:
```bash
git clone https://github.com/Varta375/ConSys.git
cd ConSys
cmake -B build -DCMAKE_PREFIX_PATH="<Ваш путь до следующих файлов/папок>/Qt/6.5.3/msvc2019_64" -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

## Тесты

```bash
cd build
ctest --output-on-failure
```
Во время тестирования температура процессора и скорость оборотов вентиляторов увеличатся

## Требования

Windows 11

Права администратора

Установленный NitroSense.exe (вентиляторы в режиме "Вручную")


## Внутренняя работа программы

Программа использует библиотеку для мониторинга температур [LibreHardwareMonitor](https://github.com/LibreHardwareMonitor/LibreHardwareMonitor?tab=readme-ov-file), обёртку для неё на C++ [lhwm-wrapper](https://gitlab.com/OpenRGBDevelopers/lhwm-wrapper) и инструмент для контроля температуры процессора [RyzenAdj](https://github.com/FlyGoat/RyzenAdj)


## Примечание к файлам проекта

Библиотека lhwm-wrapper хранится 
в `external/` в скомпилированном виде по причине специфичной сборки и чувствительности к версии LibreHardwareMonitorLib.dll

Аналогично, ryzenadj.exe и драйвер WinRing0x64.sys являются готовыми бинарными файлами, не требующими сборки
