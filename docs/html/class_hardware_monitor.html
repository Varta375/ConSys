Подробное описание

Класс для получения данных о температуре и нагрузке CPU и GPU.

Предоставляет интерфейс для чтения показателей с датчиков системы через предопределенные пути в файловой системе.

Методы

◆ GetCpuLoad()

int HardwareMonitor::GetCpuLoad ( )

Возвращает значение общей загрузки CPU.

Возвращает
Загрузка процессора в процентах (от 0 до 100). Возвращает 0, если данные еще не были обновлены, или -1 в случае возникновения ошибки.
Предупреждения
Значение актуально только после вызова метода Update().
Исключения
std::exceptionОшибки логируются в std::cerr, а в качестве результата возвращается false

◆ Update()

void HardwareMonitor::Update ( )

Обновляет информацию по всем датчикам.

Выполняет чтение текущих значений температуры и нагрузки для CPU и GPU и сохраняет их во внутренние переменные. Рекомендуется вызывать этот метод перед вызовом геттеров.
