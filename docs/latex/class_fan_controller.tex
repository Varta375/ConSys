\doxysubsection{Подробное описание}
Класс для управления вентиляторами и их мониторинга для ноутбуков Acer Nitro 5 AN515-\/45 (Ryzen 5 5600H RTX 3060). 

Управляет оборотами через запись в реестр HKLM\textbackslash{}\+SOFTWARE\textbackslash{}\+OEM\textbackslash{}\+Nitro\+Sense\textbackslash{}\+Fan\+Control и отправку структурированных команд службе PSsvc через именованный канал Идея реализации базируется на решениях проекте Nitro\+Sensual от KRWCLASSIC \begin{DoxySeeAlso}{См. также}
\href{https://github.com/KRWCLASSIC/NitroSensual}{\texttt{https\+://github.\+com/\+KRWCLASSIC/\+Nitro\+Sensual}} 
\end{DoxySeeAlso}
