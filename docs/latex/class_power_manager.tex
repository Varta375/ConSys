\doxysubsection{Подробное описание}
Класс для настройки электропитания ноутбука 

Данные получены с помощью консольной команды powercfg -\/list. Позволяет переключаться между режимами "{}Экономия энергии"{}, "{}Сбалансированный"{} и "{}Высокая производительность"{} Использует Windows API функции Power\+Set\+Active\+Scheme \begin{DoxySeeAlso}{См. также}

\end{DoxySeeAlso}
